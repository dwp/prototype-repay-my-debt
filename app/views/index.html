Enhanced DD start date selection for new plans

  • Jira ticket
Status
Ready for UR
Description
Currently, citizens using the Repay service can only select the day of the month for their Direct Debit start date, while those using the contact centre can choose a start date up to 61 days in the future at the user's request. This inconsistency limits flexibility and creates confusion for customers transitioning from benefits to employment. This should not be offered to all users, it is optional if required.
